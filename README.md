# node-dynamodb-connect

### Install bun

```bash
npm install bun -g
```

### Install dependencies:

```bash
bun install
```

### Generate tables from seed

```bash
bun run bun:db-seed <number-of-tables> <number-of-items-per-table>
```

### Export packed data to `assets` folder

```bash
bun run bun:db-export-packed-data
```
